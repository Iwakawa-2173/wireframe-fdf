# wireframe-fdf

![Демонстрация результата](https://github.com/Iwakawa-2173/wireframe-fdf/blob/main/d.gif)

## Описание
Эта программа делает плавный переход отображений друг в друга двух карт высот на основе двух картинок.

## Управление
Фигура вращается движением мышки с зажатой левой клавишей. Колёсико мыши отвечает за удаление/приближение фигуры к камере.

## Использование и запуск
Просто скачайте все файлы из репозитория и запустите программу командой:
   ```
   python game.py
   ```
